# Asteroids-Extreme
Asteroidsproject for practicing some C++ and OpenGL.
Feel free to use for your own education.

**In-game features:**
- Rocket being able to fly through space
- "Asteroids"
- UFO's
- Lasers (being able to shoot the asteroids)
- Scoring system
- Sound
- Speed and location display
- Random generated star system
